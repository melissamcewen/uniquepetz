Strange Textures

Many of the built-in game textures are well-documented elsewhere like on Moonflower's mammoth texture chart. But what about the other ones? If you examine the resource dll files with Resource Hacker, there are few bmps that I rarely see used on Petz. For reasons that will become obvious as soon as I show you them. But with creativity you could do cool things with these!

First our victim:

He has all the "natural" colors in his coat so we can test these textures out at "1" transparency to see what happens.
